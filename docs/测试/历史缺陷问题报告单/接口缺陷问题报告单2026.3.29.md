## 缺陷列表

### 缺陷 1


| 项        | 内容                                                                            |
| -------- | ----------------------------------------------------------------------------- |
| **标题**   | 重启系统后token问题                                                                  |
| **严重程度** | 致命                                                                            |
| **现象**   | 在后端服务重启后，重启前获取的token没有过期                                                      |
| **复现步骤** | 1.使用/api/auth/login登录，返回token.2.重新启动后端服务3.使用该token,调用/api/users接口，接口仍能够返回数据 |
| **期望结果** | 在重启后端后，应该使之前的token过期                                                          |
| **附件**   | 无                                                                             |
