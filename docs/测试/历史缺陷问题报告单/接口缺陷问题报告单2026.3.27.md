## 缺陷列表

### 缺陷 1


| 项        | 内容                                                                       |
| -------- | ------------------------------------------------------------------------ |
| **标题**   | 影像查询问题                                                                   |
| **严重程度** | 一般                                                                       |
| **现象**   | 查询不存在的病例时，会返回查询成功                                                        |
| **复现步骤** | 1.使用管理员用户登录获取token.2.调用GET /api/cases/9999/images3.返回"message": "查询成功" |
| **期望结果** | 查询不存在的病例时，应返回查询失败                                                        |
| **附件**   | 无                                                                        |


---

### 缺陷 2


| 项        | 内容                                                                   |
| -------- | -------------------------------------------------------------------- |
| **标题**   | 上传影响问题                                                               |
| **严重程度** | 严重                                                                   |
| **现象**   | 上传影像时，能够上传非影像类型文件                                                    |
| **复现步骤** | 1.使用管理员用户登录获取token.2.调用POST /api/casess/1/images,选择非影像类型文件上传，返回上传成功 |
| **期望结果** | 禁止非影像文件的上传                                                           |
| **附件**   | 无                                                                    |
