## 缺陷列表

### 缺陷 1


| 项        | 内容                                                                    |
| -------- | --------------------------------------------------------------------- |
| **标题**   | 多次调用登录接口问题                                                            |
| **严重程度** | 一般                                                                    |
| **现象**   | 多次调用登录接口后，旧的token没有过期，依旧可以使用                                          |
| **复现步骤** | 1.多次调用/api/auth/login，返回多个token。 2.使用旧的token,调用/api/users接口，接口仍能够返回数据 |
| **期望结果** | 每次调用/api/auth/login后，让前一次的token失效                                     |
| **附件**   | 无                                                                     |
